# EcoTrace 🌱

EcoTrace is a full-stack real-time carbon footprint tracker and environmental impact simulator. Built using React, Tailwind CSS, Express, and modern TypeScript, it helps users estimate, log, and optimize their carbon footprint with interactive estimators, habit logging, and automated metric validations.

---

## ✨ Features

- **Interactive Carbon Calculators**: Deep calibration across Transport, Energy, Diet, and Lifestyle sectors using EPA & IPCC greenhouse metrics.
- **Dynamic Habit & Offset Logging**: Real-time logging of daily actions, ecological savings, and offsets.
- **Modern Dashboard & Visualizations**: High-contrast, beautifully structured dashboard with rich interactive charts and progress tracking.
- **Express Backend with Vite Integration**: Seamless full-stack architecture running a single compiled server runtime.

---

## 🛠️ Tech Stack

- **Frontend**: React, Tailwind CSS, Lucide Icons, Recharts, Motion (Framer Motion)
- **Backend**: Express, TypeScript (`tsx` development runner, bundled to a highly-optimized single-file CJS build via `esbuild`)
- **Metadata Rules**: Integrated alignment parameters, ISO-14064 compliance baseline modeling.
